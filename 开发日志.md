1、初始化npm配置文件

# 在当前根目录下运行
$ npm init -y
执行完成之后，当前根目录下回产生package.json文件，该文件为npm的配置文件

2、安装依赖的gulp插件
# 安装gulp依赖
$ npm i -D gulp

3、 安装gulp-sass依赖
$ npm i gulp-sass -D

4、全局安装live-server启动web服务
$ npm i -g live-server
# 进入项目录
# 启动web服务
$ live-server
